Integrate ComfyUI + Flux with LibreChat

A step-by-step guide to running ComfyUI with the FLUX.1-dev model and exposing it as a Stable Diffusion compatible API through a Flask proxy, so LibreChat can generate images right from the chat.

🐋 Docker 🟩 CUDA 12.x 🐍 Python 3.11 🌐 Flask Proxy 🔥 FLUX.1-dev

◇ System Architecture

Understanding the request flow from LibreChat to ComfyUI's GPU.

🐋 LibreChat (Docker)
↓ SD API calls ↓
🔀 Flask Proxy (port 7860)
↓ ComfyUI API ↓
🎨 ComfyUI (port 8188)
↓ Inference ↓
⚡ FLUX.1-dev · RTX GPU
💡
The Flask proxy translates LibreChat's Stable Diffusion (A1111) API calls into ComfyUI's workflow-based API, bridging two different ecosystems seamlessly.

1 Install ComfyUI

Clone the repository, create a virtual environment, and install PyTorch with CUDA support.

Clone & Activate Virtual Environment

📄 Terminal — Setup
# Clone repo
git clone https://github.com/comfyanonymous/ComfyUI
cd ComfyUI

# Create venv with Python 3.11
py -3.11 -m venv venv
venv\Scripts\activate

Install PyTorch with CUDA

📄 Terminal — CUDA Install
# Install latest CUDA-enabled PyTorch
pip install --pre torch torchvision torchaudio --index-url https://download.pytorch.org/whl/nightly/cu128

Verify CUDA

📄 Terminal — CUDA Test
python -c "import torch; print(torch.cuda.is_available()); print(torch.cuda.get_device_name(0))"
Expected output:
True
NVIDIA GeForce RTX 5090
⚠️
Make sure torch.cuda.is_available() returns True. If it returns False, your CUDA setup needs attention.

Install Requirements & Start

📄 Terminal — Finish Install
# Install all requirements
pip install -r requirements.txt

# Start ComfyUI
python main.py --listen 0.0.0.0 --port 8188

4 Flask Proxy Application

A custom Flask app that translates Stable Diffusion WebUI (A1111) API calls from LibreChat into ComfyUI-compatible workflow requests.

📄 comfyui-proxy/app.py
import base64, json, requests, time, random
from flask import Flask, request, jsonify

app = Flask(__name__)
COMFYUI = 'http://host.docker.internal:8188'


def build_workflow(prompt, width=1024, height=1024, steps=20):
    """Build a ComfyUI workflow dict from a text prompt.

    Args:
        prompt: The text prompt for image generation.
        width: Image width in pixels.
        height: Image height in pixels.
        steps: Number of sampling steps.

    Returns:
        Dictionary representing the ComfyUI workflow.
    """
    seed = random.randint(0, 2**32)
    return {
        '1': {'inputs': {'ckpt_name': 'flux1-dev.safetensors'}, 'class_type': 'CheckpointLoaderSimple', '_meta': {'title': 'Load Checkpoint'}},
        '2': {'inputs': {'clip_name1': 'clip_l.safetensors', 'clip_name2': 't5xxl_fp8_e4m3fn.safetensors', 'type': 'flux', 'device': 'default'}, 'class_type': 'DualCLIPLoader', '_meta': {'title': 'DualCLIPLoader'}},
        '3': {'inputs': {'vae_name': 'ae.safetensors'}, 'class_type': 'VAELoader', '_meta': {'title': 'Load VAE'}},
        '4': {'inputs': {'text': prompt, 'clip': ['2', 0]}, 'class_type': 'CLIPTextEncode', '_meta': {'title': 'CLIP Text Encode'}},
        '5': {'inputs': {'width': width, 'height': height, 'batch_size': 1}, 'class_type': 'EmptyLatentImage', '_meta': {'title': 'Empty Latent Image'}},
        '6': {'inputs': {'noise_seed': seed}, 'class_type': 'RandomNoise', '_meta': {'title': 'RandomNoise'}},
        '7': {'inputs': {'conditioning': ['4', 0]}, 'class_type': 'FluxGuidance', '_meta': {'title': 'FluxGuidance'}},
        '8': {'inputs': {'model': ['1', 0], 'conditioning': ['7', 0]}, 'class_type': 'BasicGuider', '_meta': {'title': 'BasicGuider'}},
        '9': {'inputs': {'sampler_name': 'euler'}, 'class_type': 'KSamplerSelect', '_meta': {'title': 'KSamplerSelect'}},
        '10': {'inputs': {'model': ['1', 0], 'steps': steps, 'denoise': 1.0, 'scheduler': 'simple'}, 'class_type': 'BasicScheduler', '_meta': {'title': 'BasicScheduler'}},
        '11': {'inputs': {'noise': ['6', 0], 'guider': ['8', 0], 'sampler': ['9', 0], 'sigmas': ['10', 0], 'latent_image': ['5', 0]}, 'class_type': 'SamplerCustomAdvanced', '_meta': {'title': 'SamplerCustomAdvanced'}},
        '12': {'inputs': {'samples': ['11', 0], 'vae': ['3', 0]}, 'class_type': 'VAEDecode', '_meta': {'title': 'VAE Decode'}},
        '13': {'inputs': {'filename_prefix': 'flux_test', 'images': ['12', 0]}, 'class_type': 'SaveImage', '_meta': {'title': 'Save Image'}}
    }


@app.route('/sdapi/v1/txt2img', methods=['POST'])
def txt2img():
    """Handle text-to-image generation requests from LibreChat."""
    data = request.json
    prompt = data.get('prompt', '')
    width = data.get('width', 1024)
    height = data.get('height', 1024)
    steps = data.get('steps', 20)
    workflow = build_workflow(prompt, width, height, steps)
    r = requests.post(f'{COMFYUI}/prompt', json={'prompt': workflow})
    prompt_id = r.json()['prompt_id']

    for _ in range(120):
        time.sleep(2)
        hist = requests.get(f'{COMFYUI}/history/{prompt_id}').json()

        if prompt_id in hist:
            outputs = hist[prompt_id]['outputs']

            for _, node_output in outputs.items():
                if 'images' in node_output:
                    img_info = node_output['images'][0]
                    img_url = f"{COMFYUI}/view?filename={img_info['filename']}&subfolder={img_info['subfolder']}&type={img_info['type']}"
                    img_bytes = requests.get(img_url).content
                    b64 = base64.b64encode(img_bytes).decode()

                    return jsonify({'images': [b64]})

    return jsonify({'error': 'timeout'}), 500


@app.route('/sdapi/v1/options', methods=['GET'])
def options():
    return jsonify({'sd_model_checkpoint': 'flux1-dev.safetensors'})


@app.route('/sdapi/v1/sd-models', methods=['GET'])
def models():
    return jsonify([{'title': 'flux1-dev.safetensors', 'model_name': 'flux1-dev'}])


if __name__ == '__main__':
    app.run(host='0.0.0.0', port=7860)

6 Docker Compose Override

Add the proxy service to your existing LibreChat docker-compose configuration.

Create Override File

📄 docker-compose.override.yml
services:

  # USE LIBRECHAT CONFIG FILE
  api:
    volumes:
    - type: bind
      source: ./librechat.yaml
      target: /app/librechat.yaml

  comfyui-proxy:
    container_name: comfyui-proxy
    build:
      context: ./comfyui-proxy
    ports:
      - "7860:7860"
    extra_hosts:
      - "host.docker.internal:host-gateway"
    restart: always
💡
extra_hosts with host.docker.internal:host-gateway is critical — it allows the Docker container to reach ComfyUI running on the host machine at localhost:8188.

7 Environment Configuration

Tell LibreChat where to find the Stable Diffusion API.

📄 LibreChat/.env
SD_WEBUI_URL=http://comfyui-proxy:7860
SD_WEBUI_KEY=
⚠️
The URL uses comfyui-proxy (Docker internal name), not localhost. This is because LibreChat and the proxy are in the same Docker network.
